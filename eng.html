My programming projects

Morasko Campus mobile map app

GitHub repo

Mobile app written in Kotlin language with use of Android Studio IDE, Maps SDK and Mapbox Studio. The app was developed as my Engineering Thesis. The app provides the Morasko Campus map and will help students to find the right way - at least in the geographical meaning.
Right now the app allows to use the map, change the base map and to select every single Morasko-based department of the university, also it supports geolocation, navigation and obtaining information about some points of interest.

GitHub repositories database

GitHub repo

A database I designed to retrieve data about GitHub repositories. Along with the database, I created a Python script that connected to the GitHub API and fetched information about the most popular repositories, filtering data for those with more than 100 stars. The biggest challenge was splitting the requests in a way that didn't exceed GitHub API limits. After running the script, the database was populated with data on approximately 90,000 repositories created between the beginning of 2023 and early February 2025.

Student sim - the game

GitHub repo

A simple game written in Python with the use of pygame library.
The game is about moving the Panda-Student (graphic source) left and right using keyboard arrows and collecting the grades that are falling down. From the grades you collect - the arithmetic average is calculated. If your grades are below 3.0 - the game is over.

Liskow municipality web map

GitHub repo

The web map of Liskow municipality done with HTML, JavaScript, jQuery and Leaflet library. The map has a dropdown menu in which the type of showed object can be selected. It also tracks the cursor and shows the geographic coordinates accordingly.
